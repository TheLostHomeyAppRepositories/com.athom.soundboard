Ladda upp dina egna .mp3 och .wav filer till Homey Pro.

På Homey Pro (2016—2019) kan du även spela ljud från den interna högtalaren med hjälp av ett Flow.
På Homey Pro (tidigt 2023) kan du använda en annan app, som Sonos, för att använda dina uppladdade ljud som ett ljudbibliotek.

För att ladda upp dina egna ljud, installera appen och navigera till appens inställningar.

MP3- och WAV-filer upp till 900 KB kan laddas upp. Spela ljud i dina egna Flows, eller koppla ihop med andra appar som Sonos för att använda denna app som ditt eget ljudbibliotek.

Exempelscenarier som har testats och visat sig vara ganska användbara:
— Spela ett ringklockljud när dörrklockan ringer.
- Förvandla Homey till en gökuret! Varje timme spelar du ett gökureljud.
- När kontaktsensorn på ditt kylskåp är öppen, spela Cookie Monsters Om Nom Nom Nom.
- Spela en mistlur för att vakna upp.
