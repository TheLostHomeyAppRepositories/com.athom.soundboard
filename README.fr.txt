Téléchargez vos propres fichiers .mp3 et .wav sur Homey Pro.

Sur Homey Pro (2016—2019), vous pouvez également jouer des sons depuis le haut-parleur interne via un Flow.
Sur Homey Pro (début 2023), vous pouvez utiliser une autre application, comme Sonos, pour utiliser vos sons téléchargés comme bibliothèque sonore.

Pour télécharger vos propres sons, installez l'application et accédez aux paramètres de l'application.

Les fichiers MP3 et WAV jusqu'à 900 KB peuvent être téléchargés. Jouez des sons dans vos propres Flows, ou connectez-vous avec d'autres applications comme Sonos pour utiliser cette application comme votre propre bibliothèque sonore.

Scénarios d'exemple testés et éprouvés comme étant très utiles :
— Jouez un son de ding-dong lorsque la sonnette retentit.
- Transformez Homey en horloge coucou ! Chaque heure, faites jouer un son de coucou.
- Lorsque le capteur de contact de votre frigo est ouvert, faites jouer le "Om Nom Nom Nom" de Cookie Monster.
- Jouez un klaxon d'air pour vous réveiller.
