Sube tus propios archivos .mp3 y .wav a Homey Pro.

En Homey Pro (2016-2019), también puedes reproducir sonidos desde el altavoz interno a través de un Flow. En Homey Pro (Principios de 2023) puedes usar otra app, como Sonos, para utilizar los sonidos que subas como biblioteca de sonidos.

Para subir tus propios sonidos, instala la app y navega a los ajustes de la app.

Archivos MP3 y WAV de hasta 900 KB pueden ser subidos. Reproduce sonidos en tus propios Flows, o conéctate con otras apps como Sonos para usar esta app como tu propia biblioteca de sonidos.

Ejemplos de escenarios probados y muy útiles:
- Reproduce un sonido de ding-dong cuando suene el timbre.
- ¡Convierte a Homey en un reloj cucú! Cada hora, reproduce un sonido de cucú.
- Cuando el sensor de contacto de tu nevera esté abierto, reproduce el sonido Om Nom Nom Nom de Cookie Monster.
- Reproduce una bocina de aire para despertar.
